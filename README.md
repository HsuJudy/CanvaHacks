# Harmony Fonts

## Overview

**Harmony Fonts** is an AI-powered typography tool designed to simplify and enhance the process of font pairing. With Harmony Fonts, users can select a font that aligns with their brand or personal preference, and the tool will automatically generate complementary, contrast, and hierarchy fonts tailored for various design purposes. This ensures that every aspect of your content is visually cohesive, improving both aesthetics and readability.

## Features

- **Font Pairing Automation:** Automatically generates font pairs that are visually appealing and aligned with your design needs.
- **User Feedback Integration:** Users can provide feedback on suggested fonts, and the tool will learn to prioritize these fonts in future suggestions.
- **Customizable Selections:** Users can start by selecting a brand or favorite font, and Harmony Fonts will generate the perfect pairings.
- **Seamless Integration:** Built using Canva's UI SDK and integrated into the Canva ecosystem with the Canva App SDK.

## Technical Details

- **AI Engine:** The font pairing algorithm is powered by the Llama model, which analyzes and suggests fonts based on content and user preferences.
- **Front-End:** The user interface is built using Canva's UI SDK, providing a smooth and intuitive experience for designers.

## Installation

To use Harmony Fonts, follow these steps:

1. **Download and Install:** The tool is integrated into Canva and can be accessed directly through the Canva platform. No separate installation is required.
2. **Access the Tool:** Within Canva, navigate to the Apps section and find Harmony Fonts.
3. **Start Designing:** Select your base font, and Harmony Fonts will do the rest, providing you with perfectly paired fonts for your design.

## Usage

1. **Select a Font:** Choose a font that aligns with your brand or personal preference.
2. **Generate Pairings:** Harmony Fonts will generate a list of complementary, contrast, and hierarchy fonts.
3. **Apply to Design:** Use the suggested fonts in your design to create a visually cohesive and aesthetically pleasing result.
4. [Not implemented yet] **Provide Feedback:** If you like a particular font, give it a thumbs up, and Harmony Fonts will learn to suggest it more often in the future.
